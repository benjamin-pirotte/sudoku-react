# Sudoku react

## Technologies :
* Typescript, 
* React, React-DOM
* Redux 
* Jest (Testing)

## Features: 
* Generate new sudokus
* Change sudokus difficulties
* Give hints to solve sudokus
* Show errors when mistakes

## How to Use : 
Install packages : 
###### npm install

Build and serve (localhost:3000) : 
###### npm start

Test : 
###### npm test 

Test watch: 
###### npm test -- --watch
