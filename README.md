# Sudoku react

## Technologies :
* Typescript, 
* React, React-DOM
* Redux 
* Jest (Testing)

## Overview : 
* Generate new sudokus
* Change sudokus difficulties
* Give hints to solve sudokus
* Show errors when mistakes

## Getting Started : 
Install packages : 
`npm install`

Build and serve (port 3000): 
`npm start`

Test (watch): 
`npm test (-- --watch)`
